The SDE includes the following files (in file name order) exported in YAML format:

* [Blueprints (blueprints.yaml)](yaml_blueprints.md)
* [Certificates (certificates.yaml)](yaml_certificates.md) 
* [EVE Graphics (graphicIDs.yaml)](yaml_graphicIDs.md)
* [Groups (groupIDs.yaml)](yaml_groupIDs.md)
* [EVE Icons (iconIDs.yaml)](yaml_iconIDs.md)
* [Ship Skins (skins.yaml)](yaml_skins.md)
* [Skin Licenses (skinLicenses.yaml)](yaml_skinLicenses.md)
* [Skin Materials (skinMaterials.yaml)](yaml_skinMaterials.md)
* [Tournament Rules (tournamentRuleSets.yaml)](yaml_tournamentRuleSets.md)
* [Inventory Types (typeIDs.yaml)](yaml_typeIDs.md)
